# Roadmap

## Stage 1

- Repository audit
- Excel data dictionary review
- CodeIgniter 4 scaffold
- Shield setup
- ERP RBAC foundation
- Multi-company and multi-site schema
- AI/OCR schema and service contracts
- Initial dashboard and document upload page

## Stage 2

- Full Skote asset integration
- CRUD for setup masters
- User, role, permission, company access, and site access pages
- Active company/site switcher

Status: Skote assets, master CRUD foundation, and active company/site switcher are implemented. User access management pages are still pending.

## Stage 3

- Customer, supplier, item, UoM CRUD
- Sales Order CRUD
- Purchase Order CRUD
- Inventory movement CRUD
- Invoice CRUD

## Stage 4

- Approval workflow engine
- Audit trail integration on create/update/delete/posting
- Stock ledger and warehouse stock balance
- Posting status and transaction locking

## Stage 5

- OCR provider implementations
- AI extraction provider implementations
- Human review UI
- Document-to-transaction conversion
- Duplicate checking workflow

## Stage 6

- General Ledger, Cash Bank, AR, Costing, Production, Fixed Asset, POS expansion
- Dashboards and charts
- Automated tests
- Deployment hardening
